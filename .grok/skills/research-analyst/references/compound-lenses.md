# Compound Lenses — Three Integrated Frames

Frame every compound through the lenses that apply. A single compound may span multiple lenses.

## Lens 1: Neuropharmacology

**Scope:** Receptors, neurotransmitters, psychotropics, nootropics, CNS peptides, neuromodulators.

| Checklist | Include |
|-----------|---------|
| Mechanism | Receptor targets, downstream signaling, BBB penetration |
| CNS effects | Documented cognitive, mood, sleep, arousal effects |
| Pharmacokinetics | Half-life, metabolism, active metabolites |
| Risks | Tolerance, dependence, serotonin syndrome, mania, withdrawal |
| Guidelines vs literature | Psychiatric/medical guideline alignment |
| **Practical Guidance** | Dosing from trials, titration, timing, drug interactions, monitoring |

## Lens 2: Performance / Bodybuilding Pharmacology

**Scope:** AAS, SARMs, peptides, GH secretagogues, PEDs, ancillaries, PCT compounds.

| Checklist | Include |
|-----------|---------|
| Mechanism | AR binding, anabolic/androgenic ratio, non-AR pathways |
| Performance effects | Lean mass, strength, recovery — certainty-labeled |
| Risks | Lipids, hematocrit, HPTA suppression, organ stress, mental health |
| Bro-science vs evidence | Community doses vs trial/observational data |
| **Practical Guidance** | Dose ranges, cycle length, ancillaries, bloodwork schedule, PCT, harm reduction |

## Lens 3: Nutrition / Supplements

**Scope:** Vitamins, minerals, ergogenic aids, adaptogens, amino acids, botanicals.

| Checklist | Include |
|-----------|---------|
| Mechanism | Absorption, bioavailability, pathway effects |
| Documented effects | RCT outcomes, effect sizes, population specificity |
| Label vs evidence | Marketing claims vs published trials |
| Safety | Upper limits, interactions, contamination risk |
| **Practical Guidance** | Dosing from trials, timing (pre/post), form selection (e.g., monohydrate vs HCl), stacking context |

## Cross-lens integration

When a compound spans lenses (e.g., creatine = nutrition + performance; modafinil = neuro + performance):

1. State which lenses apply and why
2. Do not duplicate mechanism — synthesize once, note lens-specific implications
3. Practical Guidance may differ by use case — label each (**Established** / **Probable** / **Speculative**)
