# Compound Profile Output Template

Use this template for `briefing.md` when input type is **compound** or user requests compound framing.

## 1. Compound Identity
- Common names, chemical name, drug class
- Legal/regulatory status (where relevant)
- Primary lens(es) per compound-taxonomy.md

## 2. Lens Applicability
Which lenses apply and why. Cross-lens notes if multi-class.

## 3. Mechanism & Pharmacology
Receptor targets, pathways, PK/PD (half-life, metabolism, bioavailability).

## 4. Documented Effects
Certainty-labeled bullets. Effect sizes where available. Population specificity.

## 5. Documented Risks & Contraindications
Adverse events, contraindications, interactions, long-term safety gaps.

## 6. Bro-Science vs Evidence
| Claim | Source tier | Literature verdict | Certainty |
Community doses, "everyone knows", forum wisdom — tagged and adjudicated.

## 7. Guidelines vs Literature
(When applicable.) Guideline | recommendation | literature alignment | DR action if mismatch.

## 8. Practical Guidance
**Required for compound requests.** Evidence-graded actionable section:

- **Dosing** — ranges from trials/labels; titration if applicable
- **Timing** — pre/post workout, with food, circadian considerations
- **Stacks** — combinations with evidence; interaction warnings
- **Monitoring** — bloodwork, side-effect watchlist, stop criteria
- **Harm reduction** — mandatory for performance compounds
- **Not recommended when** — Unknown certainty, contraindicated populations

Each bullet tagged: **Established** / **Probable** / **Speculative** / **Unknown**

## 9. Compound Context Summary
≤150 words. Actionable educational frame — what this compound is for, what evidence supports, what to watch, bottom-line recommendation tier.

## 10. Guidance & Application Notice
Evidence-graded synthesis; user assumes responsibility; **Unknown** = no recommendation.

## 11. References
DOI/PMID/URL for all cited sources.
